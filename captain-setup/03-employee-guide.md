# 👨‍💼 دليل الموظف — تفعيل QAYDAO AI Captain

**المدة المتوقعة:** 30-45 دقيقة
**المتطلبات:** صلاحيات Administrator في Chatwoot

---

## ✅ ما تم إنجازه مسبقاً (لا تعمل عليه):

- [x] تفعيل ميزة Captain على الـ Account
- [x] إعداد OpenAI API key
- [x] فتح الواجهة (https://chat.qaydao.com/app/accounts/1/captain/assistants)

**كل ما عليك:** اتباع الخطوات أدناه بالترتيب.

---

## 📋 الخطوة 1: إنشاء Assistant

### 1.1 افتح الواجهة:
```
https://chat.qaydao.com/app/accounts/1/captain/assistants
```

### 1.2 اضغط على زر **"New Assistant"**

### 1.3 املأ الحقول:

| الحقل | القيمة |
|------|--------|
| **Name** | `QAYDAO AI` |
| **Description** | `مساعد ذكي لخدمة عملاء قيدداو خارج أوقات العمل` |
| **Instructions** | ← انسخه من ملف `01-system-prompt.md` (القسم: "محتوى الـ Instructions") |
| **Product name** | `قيدداو - QAYDAO` |
| **Welcome message** | `🤖 مرحباً، معك QAYDAO AI 🤍 كيف أقدر أساعدك؟` |

### 1.4 اضغط **Save / Create**

✅ **النتيجة المتوقعة:** يظهر الأسيستنت في القائمة.

---

## 📚 الخطوة 2: رفع Documents (المعرفة)

### 2.1 من القائمة الجانبية، اضغط **Documents**

### 2.2 اضغط **"New Document"**

### 2.3 للملف الأول، املأ:

| الحقل | القيمة |
|------|--------|
| **Name** | `سياسة-الشحن-والتوصيل` |
| **Type** | اختر `Text` (أو Custom Text) |
| **Content** | ← انسخ المحتوى من `02-faq-documents.md` → Document #1 |
| **Assistant** | `QAYDAO AI` |

### 2.4 اضغط **Save**

### 2.5 كرر نفس العملية لكل Document في الملف:

- [x] Document #1: سياسة الشحن والتوصيل
- [ ] Document #2: سياسة الاسترجاع والإلغاء
- [ ] Document #3: المنتجات والمواصفات
- [ ] Document #4: الفروع وطرق التواصل
- [ ] Document #5: طرق الدفع والفوترة
- [ ] Document #6: قواعد التحويل والحالات الخاصة

✅ **عند الانتهاء:** ستظهر 6 Documents مرفوعة، ولكل واحد حالة `Processing` ثم `Ready` خلال دقائق.

⏳ **الانتظار مهم:** Captain يحتاج 2-5 دقائق لمعالجة كل Document (vector embedding). انتظر حتى تصبح كلها `Ready` قبل المرحلة التالية.

---

## 🧪 الخطوة 3: اختبار في Playground

### 3.1 من القائمة الجانبية، اضغط **Playground** (إذا متوفر)

### 3.2 جرّب الأسئلة التالية:

| السؤال للاختبار | الرد المتوقع |
|------------------|--------------|
| `كم مدة التوصيل؟` | يجب يجيب بـ 75-90 يوم |
| `هل لكم فروع بالرياض؟` | لا يوجد، أونلاين فقط |
| `أبغى ألغي طلبي` | يحوّل لموظف |
| `زعلانة من تأخركم` | يعتذر ويحوّل فوراً |
| `كيف أرجع المبلغ؟` | يشرح + يحول لقسم المالية |

### 3.3 لو أي إجابة غير دقيقة:

1. عدّل الـ Document المرتبط من قسم Documents
2. انتظر إعادة الفهرسة (2-3 دقائق)
3. اختبر مرة أخرى

---

## 🔗 الخطوة 4: ربط Captain بـ Inbox

### 4.1 افتح Settings → Inboxes

### 4.2 اختر الـ Inbox المطلوب (نوصي البدء بـ WebWidget فقط)

### 4.3 اذهب لتبويب **Configuration** أو **Bot Configuration**

### 4.4 ابحث عن قسم **"AI Agent"** أو **"Captain Assistant"**

### 4.5 اختر **`QAYDAO AI`** من القائمة المنسدلة

### 4.6 إعدادات إضافية:

| الخيار | القيمة الموصى بها |
|--------|--------------------|
| **Active outside business hours only** | ✅ نعم (للبدء الآمن) |
| **Handoff trigger** | بعد 5 رسائل من Captain |
| **Confidence threshold** | 0.7 (لو يقل، يحوّل) |
| **Fallback agent** | شيماء أو فيّ (أو "أي شخص متاح") |

### 4.7 اضغط **Save**

### 5.2 ابحث عن **"Business Hours"** أو **"Working Hours"**

### 5.3 فعّلها وضع:

| اليوم | الساعات |

| الثلاثاء | 9:00 ص - 6:00 م |

| الخميس | 9:00 ص - 6:00 م |
| الجمعة | ❌ مغلق |
| السبت | ❌ مغلق |

### 5.4 **Time Zone:** `Asia/Riyadh`

### 5.6 اضغط **Save**

## 🎯 الخطوة 6: التشغيل التجريبي

### 6.1 جرّب من جانب العميل:
- افتح qaydao.com في متصفح خاص (incognito)
- ابدأ محادثة من WebWidget بعد الساعة 6 مساءً
- اسأل سؤال بسيط (مثل "كم مدة التوصيل؟")
- تحقق أن Captain يرد فوراً

### 6.2 جرّب التحويل:
- اكتب "أبغى أكلم موظف" أو "زعلانة"
- تحقق أن المحادثة تحوّل للفريق + label "AI Handed Off"

## 🚨 خطة الطوارئ — كيف توقف Captain فوراً؟

لو أي مشكلة طارئة:

### الطريقة 1 (الأسرع):
1. اذهب لـ Settings → Inboxes → اختر الـ Inbox
2. Bot Configuration → أزل QAYDAO AI من Assistant
3. Save → Captain يتوقف فوراً

### الطريقة 2 (إيقاف كامل):
1. Captain → Assistants → QAYDAO AI
2. اضغط `Disable` أو `Pause`

### الطريقة 3 (الطوارئ القصوى):
- تواصل مع رامي أو الفريق التقني لإيقاف على مستوى السيرفر
