### 🧠 استخراج الـ Top FAQs من بيانات مايو (تلقائياً):

```sql
-- الأسئلة المتكررة من العملاء (تحليل clusters)
SELECT 
  LEFT(content, 50) AS question_pattern,
  COUNT(*) AS occurrences
FROM messages m
JOIN conversations c ON c.id = m.conversation_id
WHERE c.created_at >= '2026-05-01' AND c.created_at < '2026-06-01'
  AND m.sender_type = 'Contact'
  AND m.message_type = 0
  AND LENGTH(content) BETWEEN 10 AND 100
GROUP BY question_pattern
ORDER BY occurrences DESC
LIMIT 100;
```

| تسريب بيانات عملاء لـ OpenAI | منخفض | عالي | OpenAI Zero Data Retention للـ enterprise / PDPL compliance |
| العميل يفقد الثقة لو عرف أنه bot | متوسط | متوسط | الشفافية: "أنا QAYDAO AI" في كل أول رد |
| Hallucination في الأسعار | متوسط | عالي | لا يعطي أسعار محددة، يوجه للموقع |
| Cost overrun | منخفض | منخفض | Daily quota في n8n: max 200 calls/يوم |
| AI يتجاوب مع طلبات ضارة | منخفض | متوسط | OpenAI moderation API قبل كل رد |
| خرق PDPL | منخفض | حرج | لا يخزن IBAN/أرقام بنكية، لا يطلبها |
| الـ AI يعطي ردود غير لائقة | منخفض | عالي | Pre-launch testing على 100+ سيناريو |

### 🛡️ PDPL Compliance Checklist:
- [ ] OpenAI Data Processing Agreement موقعة (DPA)
- [ ] Zero Data Retention enabled في OpenAI API (مدفوع)
- [ ] إشعار للعميل: "محادثتك تتم معالجتها بواسطة AI" — قبل البدء
- [ ] Opt-out: العميل يقدر يطلب "بشر فقط" ويحفظ choice
- [ ] لا تخزين IBAN/CR في prompts
- [ ] Logs encrypted في n8n + PostgreSQL
- [ ] Right to deletion (PDPL Article 7): API لحذف logs عميل محدد
